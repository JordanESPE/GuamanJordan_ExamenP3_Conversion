Presentación: Estudiante de Ingeniería de Software especializado en CI/CD. Este proyecto demuestra la implementación de funciones de conversión de temperaturas y cálculo de promedios móviles con pruebas automatizadas, integración continua con GitHub Actions y despliegue automatizado en Firebase Hosting.

Funcionalidades del Proyecto

  • Conversión Fahrenheit a Celsius: Convierte temperaturas de °F a °C usando la fórmula C = (F - 32) × 5/9
  • Conversión Celsius a Fahrenheit: Convierte temperaturas de °C a °F usando la fórmula F = C × 9/5 + 32
  • Promedios Móviles: Calcula promedios móviles de series numéricas con ventana deslizante
  • Validaciones Completas: Incluye manejo robusto de errores (TypeError y RangeError)
  • Cobertura de Pruebas: >85% de cobertura en líneas y ramas

Ejemplo de Promedios Móviles

Serie de entrada: [10, 20, 30, 40]

Ventana: 2

Resultado: [15.00, 25.00, 35.00]


Cálculo:
• Promedio(10, 20) = 15.00
• Promedio(20, 30) = 25.00
• Promedio(30, 40) = 35.00
